# 기능 설명
1. 버튼을 누르면 홈화면으로 이동되는데, 사용자의 클릭 값을 저장
2. 다시 앱으로 돌아와 기록 중지 및 저장을 누르면 저장됨
3. 마지막으로 저장된 기록을 바탕으로 사용자의 클릭을 동일하게 수행
